# BMI Calculation Program (Java)

このプログラムは、教科書第2章の学習として作成した **BMI（Body Mass Index）計算プログラム** です。  
Javaの基本構文（クラス、メソッド、コンストラクタ、if文、入出力）を学ぶ目的で制作しました。

---

## 📘 概要
ユーザーから **体重（kg）** と **身長（m）** を入力として受け取り、  
BMI（体格指数）を計算し、判定結果（低体重・普通・肥満）を表示します。
